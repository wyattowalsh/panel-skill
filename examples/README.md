# Example Panel Discussions

This directory contains example panel discussions demonstrating different
topic types and skill configurations.

## Examples

| File | Topic Type | Complexity | Key Features |
|------|------------|------------|--------------|
| [architecture-decision.md](architecture-decision.md) | Technical | Medium | Trade-off analysis, phased recommendations |
| [security-implementation.md](security-implementation.md) | Technical | High | Deep expertise, risk assessment |
| [business-strategy.md](business-strategy.md) | Strategic | High | Multiple stakeholders, values tension |

## Using Examples

These examples demonstrate:

1. **Diversity scoring** in action
2. **Contrarian protection** preserving minority views
3. **Confidence-weighted synthesis** in final reports
4. **Complexity-appropriate** panel sizes

## Running Your Own

```bash
# Install the skill
npx skills add wyattowalsh/panel-skill

# Run a panel discussion
/panel "Your topic here"

# With configuration
/panel size:5 depth:deep "Complex strategic question"
```
